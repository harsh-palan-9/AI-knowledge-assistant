1. User uploads document
2. Document is chunked
3. Chunks converted to embeddings
4. Stored in Pinecone (per user)
5. User asks question
6. Question converted to embedding
7. Relevant chunks retrieved
8. Passed to LLM
9. LLM generates answer
10. Response returned to UI
